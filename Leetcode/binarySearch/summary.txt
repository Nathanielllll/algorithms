本题的二分查找的思路来源于二分查找的基本思想（应用）：查找一个有范围的整数，关键在于利用单调性逼近这个整数。「力扣」上很多问题都基于本题设计，属于「使用二分查找最大值最小化」的一类问题的例题。

1.标准二分查找三大框架之一
Binary Search有三个框架
（1）寻找符合条件的最左端的那个
（2）寻找恰好符合条件的那个
（3）寻找符合条件的最右端的那个

2.本题就是用的第一个框架 ----找符合条件的最左端的那个（最小的那个）-----正合题意
框架：

L = 最小的可能值；
R = 最大的可能值；
while (L < R)
{
    mid = (L + R) >> 1;
    if (检查发现，mid是ok的)
        R = mid;
    else
        L = mid + 1;
}
return L;




求精确的，用while (L <= R)
求最左边/最右边，用while (L < R)
